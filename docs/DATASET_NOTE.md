# Dataset Note

## Dataset Role
This project uses a malicious URL dataset for four-class classification:
- benign
- defacement
- malware
- phishing

## Repository File
The working dataset file included in this repository is:

`data/malicious_phish_CSV.csv`

## Purpose
The dataset is used for:
- preprocessing
- feature extraction
- training shallow and deep models
- evaluating final pipeline performance

## Notes
- The main task is multi-class malicious URL detection
- URLs are classified based on their label
- The project evaluates both individual stages and the final integrated pipeline

## Recommendation
For future improvement, the repository can include:
- a smaller public sample dataset for quick testing
- a separate note describing full dataset source and preprocessing steps
