# Contributing to Neural Sieve Cascade (NSC)

Thank you for your interest in contributing to this project.

Neural Sieve Cascade (NSC) is a research-oriented malicious URL detection project built around a confidence-driven three-stage pipeline using:
- Logistic Regression + TF-IDF
- CNN / LSTM / BiLSTM ensemble
- TinyBERT

This repository is primarily maintained as a project and portfolio repository, but improvements, fixes, and suggestions are welcome.

---

## Ways to Contribute

You can contribute by helping with:

- bug fixes
- README and documentation improvements
- notebook cleanup and code refactoring
- modularizing notebook logic into reusable Python scripts
- improving reproducibility
- evaluation scripts and metrics reporting
- inference utilities for single-URL prediction
- deployment support such as API or demo integration

---

## Before You Start

Please first:
1. read the `README.md`
2. check existing issues or open a new discussion
3. make sure your contribution matches the scope of the project

For major changes, it is better to open an issue first and describe:
- what you want to improve
- why the change is useful
- how you plan to implement it
